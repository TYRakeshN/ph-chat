# Getting Started with PhChat

Minimal real-time chat system built with pure Reactjs[FE],NodeJS & Express.

## Installation

Clone the repository and use the following command to install the dependencies.

```bash
npm install
```

## Usage

```bash
npm start
```
## live demo link
